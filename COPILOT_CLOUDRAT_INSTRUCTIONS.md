# Managed Hosting (External Service)

This repository contains only the **open-source, self-hosted** Real Analytics Tracker (RAT).

A separate managed hosting service may be offered at [cloudrat.io](https://cloudrat.io) or similar. That service is not part of this repository. If you prefer managed hosting instead of self-hosting, please refer to that provider's documentation and terms.

For contributing to or using the open-source RAT software, see [CONTRIBUTING.md](CONTRIBUTING.md) and [README.md](README.md).
